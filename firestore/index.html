Step 3: Send Notifications

This webpage is used to send notifications to all users.Make sure and update the notifications page before sending new notification. When users tap on new notifications it will go to notifications page on app.

  1. Click on this link
  2. Click New Message
  3. Type notification in Message Text field
  4. Select Send Now or Later
  5. Select Target User Segment
  6. Select dropdown Select App and select com.aquaaerobic.aquaguide
  7. Click Send Message
